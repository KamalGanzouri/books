# Books Management System

A simple Spring Boot application for managing a collection of books. This project serves as a basic example of building a RESTful backend using Spring Boot and Maven.

## Features

- Add, update, delete, and retrieve books
- RESTful API design
- Maven wrapper included for easy project setup

## Technologies Used

- Java 17+
- Spring Boot
- Maven
- Lombok (for boilerplate reduction)


## How to Run the Project

1. **Clone the repository:**

   ```bash
   git clone https://github.com/KamalGanzouri/books.git
   cd books
    ```
2. **Run the application using Maven Wrapper:**
    ```bash
    ./mvnw spring-boot:run
    ```
   **Or on Windows:**
    ```bash
    mvnw.cmd spring-boot:run
    ```
3. **Prerequisites:**

- Java (JDK 17 or later)

- Maven (or use the provided Maven Wrapper)

This will start the Spring Boot application.
